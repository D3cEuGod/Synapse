# Synapse Employee Wellbeing Platform

Synapse is an employee wellbeing platform developed as a 12-week software engineering group project for FDM Group.

The platform supports employee wellbeing through physical activity tracking, mental health resources, peer support, rewards, and role-based dashboards.

## My Contribution

## Key Features

- Employee profile and identity management
- Physical wellbeing tracking
- Mental health support resources
- Peer support and booking features
- Gamification and rewards
- HR-facing analytics/prototype views
- Real-time notification concept using Firestore

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript (ES modules)
- Firebase Authentication
- Firebase Firestore
- Chart.js
- Lucide Icons
